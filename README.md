# Weather AI Agent Pro

Portfolio-ready starter using Flask + LangChain + xAI.

## Features

-   xAI Grok via OpenAI-compatible endpoint
-   ReAct agent
-   OpenWeatherMap tool
-   DuckDuckGo search
-   Responsive chat UI

## Setup

1.  python -m venv .venv
2.  pip install -r requirements.txt
3.  copy .env.example to .env
4.  add keys
5.  python app.py
